这是一个基于**层次分析法(AHP)**构建的**优秀党员综合评价系统**，使用Excel实现，适用于高校或党组织对党员发展对象进行多维度、定量化的考核与评估。

## 🎯 项目动机

本项目源于一个真实的工作场景：在担任辅导员学生助理期间，我观察到党员发展对象的评价工作缺乏一个**标准化、可量化的工具**。评价过程往往依赖主观经验，难以在不同候选人之间进行公平、一致的比较。

为了提升评价工作的**效率与科学性**，我自发研究了决策分析领域的经典方法——层次分析法(AHP)，并利用广泛可用的Excel平台，在**两天内**完成了从理论研究到工具实现的全过程。这个工具成功地将定性的评价准则转化为定量的决策模型，帮助老师更清晰、高效地完成考评工作。

## 💡 核心价值

将复杂的多准则决策问题，通过系统性的方法转化为**易于操作、结果透明**的实用工具，实现：
- **评价标准化**：统一评价尺度，确保公平性
- **过程透明化**：权重计算全程可追溯
- **决策科学化**：基于数据而非主观经验

## ✨ 项目特点

- **📊 层次化评价体系**：包含5个一级指标、11个二级指标、24个三级指标，全面覆盖党员发展关键维度
- **⚖️ 自动权重计算**：基于判断矩阵自动计算各层指标权重，支持动态调整与实时更新
- **✅ 一致性检验**：自动进行CR值检验，确保判断矩阵逻辑合理（所有CR < 0.1）
- **📈 总分合成**：自动汇总各层级权重，生成综合评价得分，便于排序与筛选

## 🔧 技术实现

### AHP模型架构
- **目标层**：评选优秀党员
- **准则层**：5个一级指标（政治素质、道德品质、现实表现、学习创新、群众基础）
- **子准则层**：11个二级指标与24个三级指标，形成完整评价树

### Excel工程化实现
- **权重计算**：采用特征值法，通过`列归一化` → `行求平均`计算权重向量
- **数据关联**：使用`VLOOKUP`函数动态调用各层级权重值
- **权重合成**：三级指标权重 = 三级权重 × 二级权重 × 一级权重
- **动态重算**：开启Excel"自动重算"，实现数据修改实时更新

### 一致性检验
- 计算一致性比率`CR = CI / RI`，当`CR < 0.1`时通过检验
- 本项目所有判断矩阵均通过一致性检验，确保逻辑合理性

## 🚀 使用方法

1. **打开文件**：用Microsoft Excel打开 `AHP_公式完成.xlsx` 文件
2. **启用自动重算**：确保Excel的**"自动重算"**功能已开启（文件 → 选项 → 公式 → 计算选项 → 自动重算）
3. **查看结果**：在 **"总权重表格"** 工作表中查看各层级权重和综合评价结果
4. **自定义调整**：在各层级工作表中调整判断矩阵数值，适应具体需求
